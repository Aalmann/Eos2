Robot

A robot can change stones within a world by placing new stone or removing existing ones. Before it can start, the robot must enter a world.

Methods

step()

The robot tries to do a step forward.

step(nr:Integer)

The robot tries tod o nr steps forward.

stepBack()

The robot tries to do a step backward.

stepLeft()

The robot tries to do a side step to the left.

stepRight()

The robot tries todo a side step to the right.

isObstacle()

Checks, if a step forwrad is impossible.

isLeftObstacle()

Checks, if a side step left is impossible.

isRightObstacle()

Checks, if a side step right is impossible.

isBackObstacle()

Checks, if a step backward is impossible.

fly()

The robot tries to fly forward.

flyBack()

The robot tries to fly backward.

flyLeft()

The robot tries to side fly to the left.

flyRight()

The robot tries to side fly to the right.

flyDown()

The robot tries to fly down.

flyUp()

The robot tries to fly up.

isFree()

Checks, if the robot can fly forward.

isLeftFree()

Checks, if the robot can fly to the left.

isRightFree()

Checks, if the robot can fly to the right.

isBackFree()

Checks, if the robot can fly backward.

isUpFree()

Checks if the robot can fly up.

isDownFree()

Checks if the robot can fly down.

turnLeft()

The robot turns 90 degrees to the left.

turnRight()

The robot turns 90 degrees to the right.

turnAround()

The robot turns around 180 degree.

drop()

The robot drops a stone on the column before her.

pickup()

The robot picks up the top stone from the column in front of her.

pickup(nr:Integer)

The robot picks up nr stones from the column in front of her.

placeStone(n:Integer)

The robot places a stone in the column in front of her. The argument n defines the relative height. n = 0 in front of her feet. n=2 in front of her head.

removeStone(n:Integer)

The robot removes a stone from the column in front of her. The argument n defines the relative height. n = 0 in front of her feet. n=2 in front of her head.

setStoneColor(color:Color)

Defines the color of new placed stones.

setMark()

Marks the column the robot stand in.

removeMark()

Removes the mark from the column the robot stands in.

isMarked()

Checks, if the robot stands in a marked column.

isStone()

Checks, if there is a stone in front of the robot.

isStone(nr:Integer)

Checks, if there are nr stones in front of the robot.

isSouth()

Checks, if the robot faces south.

isWest()

Checks, if the robot faces west.

isNorth()

Checks, if the robot faces north.

isEast()

Checks, if the robot faces east.

Example

world:World
world.load("labyrinth.eow")
cute:Robot
world.enter(cute)
repeat while not cute.isMarked()
  cute.turnLeft()
  repeat solange cute.isObstacle()
    cute.turnRight()
  *repeat
  cute.step()
*repeat
