World

A world is required to use a robot. In order to use a world you should created it beforehand with the world editor and load it. If you don't do that the world will be empty with a single entrance.

A world can contain any numbe rof entrances. These are the starting points for the robots. Everytime a robot enters a world, it is placed at the first free entrance.

Worlds are creted and save with the world editor. A robot program can load such worlds. For it work reliably, you have to save your work in the same folder that contains the world.

Methods

enter(robot:Robot)

Adds a robot to that world. This only works once per entrance, if you use the default world once.

setStone(x:Number, y:Number, z:Number)

Places a stone at the position.

setStoneColor(color:Color)

Sets the color of new stones.

dropStone(x:Number, y:Number)

Drops a stone at the top of the column.

removeStone(x:Number, y:Number, z:Number)

Removes the stone from that position.

pickupStone(x:Number, y:Number)

Picks up the top stone of that column.

load(file:String)

Loads a world from a file, that has been created with the world editor.

litterStones(left:Integer,top:Integer,right:Integer,bottom:Integer,density:Real)

Litters the rectangle randomly with stones.The density defines how much is covered.

Example

w:World
eos:Robot
w.load("labyrinth.eow")
w.enter(eos)
eos.step()
