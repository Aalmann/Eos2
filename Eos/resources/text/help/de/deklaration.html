Deklaration

Eine Deklaration dient dazu eine Variable zu definieren. In EOS wird implizit automatisch auch ein Objekt erzeugt und in dieser Variable gespeichert. Diese Regel hat die Ausnahme, dass die Klassen Figur, StrichFigur und GefüllteFigur nur Sammelbegriffe für andere Klassen sind. Hat eine Variable oder Parameter den Typ Figur, kann sie zwar jedes Figur Objekt aufnehmen, ist anfänglich jedoch leer. Greift man auf so eine Variable zu, bevor man ein anderes Objekt zugewiesen hat, gibt es einen Fehler.

Enthält das gesamte Programm keine Variable der Klasse Fenster, aber mindestens eine Figur, wird automatisch ein geheimes Fenster Objekt erzeugt. Jedes neue Objekt wird automatisch diesem Fenster hinzugefügt.

Syntax

Name:Klasse

Erzeugt eine Variable mit diesem Namen, welche ein Objekt der Klasse enthält.

Name_1, Name_2, ..., Name_n:Klasse

Erzeugt mehrere Variablen der selben Klasse auf einmal.

Beispiel

augeLinks, augeRechts:Ellipse
mund:Rechteck
gesicht:Kreis
nase:Dreieck
