Eigene Methoden

Eine eigene Methode ist eine Anweisungssequenz, welche man immer wieder aufrufen kann. Man spart sich auf diese Weise wiederkehrende Sequenzen immer wieder tippen zu müssen. EOS kennt Methoden und Anweisungen. Methoden können auf alle definierten Objekte zugreifen. Anweisungen können ausschliesslich Parameter verwenden.

Mehrere Methoden müssen sich zumindest im Namen oder in ihrer Anzahl der Parameter unterscheiden.

Schreibweise

Methode ohne Parameter

methode Name
  Sequenz
*methode

Methode mit Parameter

methode Name( Parameter )
  Sequenz
*methode

Methode mit Parameter und Ergebnis

methode Name( Parameter ):Typ
  Sequenz
  ergebnis = ...
*methode

Parameter

Parameter schreibt man in der Form

Name : Typ, Name : Typ , ... , Name : Typ

Hat man mehrere Parameter die den selben Typ haben, kann man folgende abkürzende Schreibweise verwenden.

Name, Name, ... , Name : Typ

Verwenden

Um eine Anweisung oder Methode aufzurufen schreibt man

Name( Argumente )

Beispiel

methode bewegen(r:RECHTECK, grad:real)
   r.Verschieben(cos(grad), sin(grad))
*methode
r1:RECHTECK
r2:RECHTECK
bewegen(r1, 10)
bewegen(r2, 20)

BNF

Methode := "methode" Name [ Parameter ] [ ":" Typ ] Sequence "*methode"
Parameter := "(" [ Name ( "," Name )* ":" Typ ( "," Name ( "," Name )* ":" Typ )* ] ")"
Name := Ein beliebiger Name der mit einem Buchstaben beginnt und keine Leerzeichen enthält
Sequenz := Eine Anweisungssequenz
Legende:
[] optionaler Teil
()* beliebig oft
