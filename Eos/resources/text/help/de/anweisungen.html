Eigene Anweisungen

Eine eigene Anweisung ist eine Anweisungssequenz, welche man mehrfach aufrufen kann. Man spart sich auf diese Weise wiederkehrende Sequenzen immer wieder tippen zu müssen. EOS kennt Methoden und Anweisungen. Methoden können auf alle definierten Objekte zugreifen. Anweisungen können ausschliesslich Parameter verwenden. Als Konsequenz steht einer Anweisung das automatisch erzeugte Fenster nicht zu Verfügung. Figuren die dort erzeugt werden, müssen manuell einem Fenster oder Gruppe hinzugeügt werden. Ansonsten bleiben Sie unsichtbar.

Mehrere Anweisungen müssen sich zumindest im Namen oder in ihrer Anzahl der Parameter unterscheiden.

Schreibweise

Anweisung ohne Parameter

anweisung Name
  Sequence
*anweisung

Anweisung mit Parameter

anweisung Name( Parameter )
  Sequence
*anweisung

Anweisung mit Parameter und Ergebnis

anweisung Name( Parameter ):Typ
  Sequence
  ergebnis = ...
*anweisung

Parameter

Parameter schreibt man in der Form

Name : Typ, Name : Typ , ... , Name : Typ

Hat man mehrere Parameter die den selben Typ haben, kann man folgende abkürzende Schreibweise verwenden.

Name, Name, ... , Name : Typ

Verwenden

Um eine Anweisung oder Methode aufzurufen schreibt man

Name( Argumente )

Beispiel

anweisung bewegen(r:RECHTECK, grad:real)
  r.Verschieben( 100 *cos(grad),  100 *sin(grad))
  r.drehen(grad)
*anweisung
i:Integer
wiederhole  36  mal
  r:RECHTECK
  bewegen(r,  10 *i)
  i = i +  1
*wiederhole

BNF

Anweisung := "anweisung" Name [ Parameter ] [ ":" Typ ] Sequence "*anweisung"
Methode := "methode" Name [ Parameter ] [ ":" Typ ] Sequence "*methode"
Parameter := "(" [ Name ( "," Name )* ":" Typ ( "," Name ( "," Name )* ":" Typ )* ] ")"
Name := Ein beliebiger Name der mit einem Buchstaben beginnt und keine Leerzeichen enthält
Sequenz := Eine Anweisungssequenz

[] optionaler Teil
