Variablen

Variablen sind die Grundlage der Programmierung. Sie enthalten die Objekte mit denen man interagiert. Man kann auch Variablen definieren die Zahlen und andere Werte enthalten.

Schreibweise

name:Typ

Typen

  • Boolean
  • Real
  • Integer
  • Zeichenkette
  • Farbe
  • Randart
  • Füllart
  • Fenster
  • Figur
  • Strichfigur
  • Linie
  • GefüllteFigur
  • Dreieck
  • Quadrat
  • Rechteck
  • Textfeld
  • Kreis
  • Ellipse
  • Gruppe
  • Turtle

Methoden aufrufen

Um eine Methode aufzurufen bzw. eine Nachricht an das Objekt zu schicken schreibt man.

name.methode(Argumente)

Attribute lesen

Um auf Attribute lesend zuzugreifen schreibt man.

name.variable

Attribute schreiben

Um Attibute zu verändern schreibt man.

name.variable := Wert

oder

name.variable = Wert

Beispiel

x:Integer
text:Zeichenkette
rad:Kreis
stift:Turtle

BNF

Deklaration := name ":" Typ
