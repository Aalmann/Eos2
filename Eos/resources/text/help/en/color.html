Color

There are the following predefined values:

yellow, red, green, blue, white, black, brown, lightblue, lightgreen, grey, lightgrey

A color can be alternatively defined using HTML codes. #RRGGBB or #RRGGBBAA. The letter have the meaning R for red, B for blue, G for green and A for transparency. All values are within the range from 00 to FF.

Usage

The color is needed for the following methods and properties.
  • Window.setGridColor(color:Color)
  • Window.getGridColor():Color
  • Window.gridColor:Color
  • Window.setBackgroundColor(color:Color)
  • Window.getBackgroundColor():Color
  • Window.backgroundcolor:Color
  • FilledFigure.setFillColor(color:Color)
  • FilledFigure.getFillColor():Color
  • FilledFigure.fillcolor:Color
  • Group.setBorderColor(color:Color)
  • Group.setFillColor(color:Color)
  • Line.setColor(color:Color)
  • Line.setColor(color:Color)
  • Line.linecolor:Color
  • LineFigure.setBorderColor(color:Color)
  • LineFigure.getBorderColor():Color
  • LineFigure.borderColor:Color
  • Textfield.setTextColor(color:Color)
  • Textfield.getTextColor():Color
  • Textfield.textColor:Color
  • Plotter.setColor(color:Color)

Example

c:Circle
c.setFillColor(red)

BNF

Color := "yellow" | "red" | "green" | "blue" | "white" | "black" | "brown" | "lightblue" | "lightgreen" | "grey" | "lightgrey"
