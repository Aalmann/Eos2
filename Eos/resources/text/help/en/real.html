Floating points numbers

The datatyp real is nearly the same as real numbers in math. There are some differences. The datatype real is rounded at about 17 digits, therefore results can never be exact. There is a theoretical maximal and minimal value, which are that big and small that it should never really matter.

Other programing languages often call that datatype double.

Syntax

You write real numbers like you normally would, the letter E is a shortcut for times 10 high.

3.1415
1.235E2 = 123.5

Example

c:Circle
c.move(4.2,3.1)

BNF

real := (digit)* ["."] (digit)* [exponent]
exponent := ("e"|"E") [("+"|"-")] (Ziffer)+
digit := "1"|"2"|"3"|"4"|"5"|"6"|"7"|"8"|"9"|"0"
