Variables

Variables are the base for programming. They store the data and object you use. Every variable will be initialised. For class datatypes a new object will be created if possible.

Syntax

name:type

Types

  • Boolean
  • Real
  • Integer
  • String
  • Color
  • BorderStyle
  • FillStyle
  • Window
  • Figure
  • LineFigure
  • Line
  • FilledFigure
  • Triangle
  • Square
  • Rectangle
  • Textfield
  • Circle
  • Ellipse
  • Group
  • Plotter

Calling methods

In order to call a method (send a message to that object), you write

name.method(arguments)

Reading properties

In order to read a property value, you write

name.property

Writing property

In order to change a property, you write

name.property := value

or

name.property = value

Example

x:Integer
text:String
wheel:Circle
pen:Plotter

BNF

Declaration := name ":" Type
