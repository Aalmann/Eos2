Userdefined methods

A userdefined method is a command sequence, that can be called several times. That way you can reuse command sequences multiple times without tipping them again. EOS knows methods and procedures. Methods can access all defined variables. Procedures can only access parameters.

Different methods need at least different names or a different count of parameters.

Syntax

Method without parameter

method name
  sequence
*method

Method with parameters

method name( parameters )
  sequence
*method

Method with parameters and result

method name( narameters ):type
  sequence
  result = ...
*method

Parameters

Parameter are written like

name : type, name : type , ... , name : type

you can define several parameters at once, that use the same type

name, name, ... , name : type

Usage

In order to invoke a procdure or method write

name( arguments )

Example

method move(r:Rectangle, degree:real)
   r.move(cos(degree), sin(degree))
*method
r1:Rectangle
r2:Rectangle
move(r1, 10)
move(r2, 20)

BNF

Method := "method" Name [ Parameter ] [ ":" Typ ] Sequence "*methode"
Parameter := "(" [ Name ( "," Name )* ":" Typ ( "," Name ( "," Name )* ":" Typ )* ] ")"
Name := Any name that starts with a letter without empty spaces
Sequence := A command sequence

[] optional
