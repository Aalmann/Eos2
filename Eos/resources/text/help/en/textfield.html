Textfield

Textfields own all properties and methods of the classes Figure, LineFigure, FilledFigure and Rectangle.

Setter- and Getter-Methods

For every property generally there are two methods for setting and getting the property value.
eg x:Number ↔ getX():Number ↔ setX(x:Number)
They are not extra documented.

Methods

appendLine(line:String)

Appends the text as new line.

deleteLine()

Deletes the last line.

Properties

textColor:Color

Color of the text.

textSize:Number

Size of the text.

font:String

The font name.

alignmentVertical:Alignment

The vertical alignment of the text.

alignmentHorizontal:Alignment

The horizontal alignment of the text.

autoAdjustSize:Booleant

Determines if the rectangle should be automatically be adjusted to the text size. This option is active as default.

hideBorder:Boolean

Determins if the border of the rectangle should be drawn.
