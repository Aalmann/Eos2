Expressions

An expression is a value or any other expression that delivers a value if evaluated. At any place where a value is needed you can use an expression that calculates that value.

In an expression you can use several operations, comparisions and functions.

Arithmetic

The common calculation types combine two numbers into a result.
  • * multiply
  • / divide
  • + add
  • - substract

Comparison

A comparison is a relation between two numbers. The result is a logical value. It can be true or false.
  • > greater than
  • < smaller than
  • >= greater equals
  • <= smaller equals
  • <> unequal
  • = equal

Logical Operation

The logical operation combines two logical values into one. Allowing complexe logical expressions.
  • and true if both values are true
  • or true if at least one value is true
  • not inverts the value

Braces

Every expression can be placed within braces in order to highlight or change the evaluation order. The evaluation of expressions follows basic algebraic rules. But if you are not sure in which order an expression is evaluated use brakets.
Remark: squared and curly braces have a totally different meaning.

Example

c:Circle
repeat while c.x < 500 und c.y >  100
   c.x := c.x *  2  +  1
*repeat
