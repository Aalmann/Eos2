Conditional Statement

Conditional statements are used to execute command if certain conditions are met or to choose between two alternatives.

Syntax

Conditional statement

if condition then
  sequence
*if

Alternative statement

if condition then
  sequence
else
  sequence
*if

Example

c:Circle
if c.x >  10  then
  c.move( 1 , 2 )
*if
if k.x <  100  then
  c.move( 1 ,  0 )
else
  c.move(- 1 ,  0 )
*if

BNF

ConditionalStatement :=
   "if" condition "then" sequence "*if" |
   "if" condition "then" sequence "else" sequence "*if" |
sequence := A sequence of statements
