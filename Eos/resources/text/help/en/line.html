Line

The line is a segment. Lines own all properties and methods of the classes Figure and LineFigure.

Setter- and Getter-Methods

For every property generally there are two methods for setting and getting the property value.
eg x:Number ↔ getX():Number ↔ setX(x:Number)
They are not extra documented.

Methods

setPoint1(x:Number,y:Number)

Sets the first point to (x,y).

setPoint2(x:Number,y:Number)

Sets the second point to (x,y).

setPoints(x1:Number,y1:Number,x2:Number,y2:Number)

Sets both points to (x1,y1) and (x2,y2).

Properties

linewidth:Number

Width of the line (thickness).

linestyle:LineStyle

The style the line is drawn in.

linecolor:Color

The color the line is drawn in.

x1:Number

The x coordinate of the first point.

y1:Number

The y coordingate of the first point.

x2:Number

The x coordinate of the second point.

y2:Number

The y coordinate of the second point.

Example

l:Line
l.setPoints(20,20,30,10)
l.setLineStyle(dashed)
l.setLineWidth(4)
l.x1 = 5
