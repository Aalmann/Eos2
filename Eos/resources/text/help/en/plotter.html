Plotter

Plotters own all properties and methods of the class Figure.

A plotter can be controlled with several methods. Every time it moves it draws a line, which allows to draw freely.

In principal a plotter draws polygons. Everytime a drawing property is changed a new polygon starts. This is important to know, in the case you change the filling style to none transparent. As every change will start a new polygon, you have to set all properties at the beginning of an area.

Setter- and Getter-Methods

For every property generally there are two methods for setting and getting the property value.
eg x:Number ↔ getX():Number ↔ setX(x:Number)
They are not extra documented.

Methods

penDown()

Starts drawinw. From now on every movement will create a line segment. Implicitly it will start a new polygon.

penUp()

Stops drawing until penDown() is called.

turnLeft(angle:Number)

Turns the plotter angle degree to the left.

turnRight(angle:Number)

Turns the plotter angle degree to the right.

left(angle:Number)

Turns the plotter angle degree to the left.

right(angle:Number)

Turns the plotter angle degree to the right.

setPenXY(x:Number,y:Number)

Moves the plotter to that position.

back(abstand:Number)

Moves the plotter backward.

forward(abstand:Number)

Moves the plotter forward.

Properties

x:Number

The x coordinate of the plotter.

y:Number

The y coordinate of the plotter.

course:Number

The course of the plotter as angle in degree.

penVisible:Boolean

Tells if the pen is visible (littel triangle) or not.

Example

p.Plotter
p.forward(10)
p.setColor(red)
p.setFillStyle(filled)
p.turnRight(60)
repeat 6 times
  p.forward(5)
  p.turnLeft(60)
*repeat
p.setPenVisible(false)
g:Group
repeat 5 times
  g.copy(p)
  g.turn(60)
*repeat
g.consume(t)
repeat forever
  g.turn(1)
*repeat
