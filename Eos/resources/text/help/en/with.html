With

The with section allows to avoid typing. If you want to call a lot of methods of the same object, you can start a section with that object. Within this section you can call methods without adding the object.

Syntax

with variable
  command sequence
*with

Example

c:Circle
with c
  move(3,2)
  stretch(2)
*with

BNF

With := "with" expression Sequence "*with"
Sequence := A sequence of commands
