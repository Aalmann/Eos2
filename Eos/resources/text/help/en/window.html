Window

The window is needed to show figures. If you do not create a window object by yourself, the system will create own for you atm you create any figure object. Every figure object will be automatically added to that window. You only have to create your own window if you want to call window methods. But then you have to add every figure by yourself.

Every unit is millimeter (mm) (25.4mm = 1 inch). EOS will trust you operation system. If your os delivers wrong dpi (dots per inch) values, the figures will have the wrong size. You can assume the values will be accurate with an error of about 10%.

Setter- and Getter-Methods

For every property generally there are two methods for setting and getting the property value.
eg x:Number ↔ getX():Number ↔ setX(x:Number)
They are not extra documented.

Methods

move(dx:Number,dy:Number)

Moves the Window.

gridOn()

Activates the grid. By default the grid is active with 10mm per line.

gridOff()

Deactivates the grid.

draw(figure:Figure)

The figure is added to that window and will now be draw. If a figure is added to another window it will change to the second. It will no longer be visible in the first window.

Properties

gridColor:Color

Color of the grid.

backgroundcolor:Color

Color of the background.

height:Number

Height of the widnow in mm.

width:Number

Width of the window in mm.

top:Number

Distance between the upper border to upper border of the screen in mm.

left:Number

Distance between the left border of the window to the left border of the screen in mm.

name:String

Title of the window.

gridwidth:Number

Distance between two gridlines. Default is 10mm.

Example

w:Window
w.setGridColor( blue )
w.setBackgroundColor( red )
w.move(- 50 , 0 )
w.setWidth( 100 )
w.setHeight( 100 )
r:Rectangle
w.draw(r)
