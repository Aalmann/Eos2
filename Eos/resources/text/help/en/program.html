Main program

The statement program is useful to separate the main program from the procedure and method declaration parts. Every command outside will automatically be added to the main program. Therfore the program statement has only optical purpose.

Syntax

program
  command sequence
*program

Example

program
  c:Circle
  c.move(3,2)
*program

BNF

Program := "program" sequence "*program"
sequence := A sequence of commands
